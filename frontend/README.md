# Frontend Application

## Tech Stack & Requirements

- Node.js >= 18.0.0
- React 19
- Vite 5.x
- TailwindCSS 4.0

## Setup & Installation

1. Navigate to the frontend folder:

```
cd frontend
```

2. Install dependencies:

```
npm install
```

3. Start the development server:

```
npm run dev
```

The app will be available at `http://localhost:5173`
